# remark-plain-text
This is a plugin for the [remark](https://github.com/remarkjs/remark) which **extracts plain text**

* in: `**bold**`
* out: `bold`


## How to use
> npm i -S remark remark-plain-text  
> npx remark -u plain-text 

or 

> npm i -g remark  
> npm i -S remark-plain-text  
> remark -u plain-text
